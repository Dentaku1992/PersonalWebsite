The expo

WetenschapsExpo Sciences is a science contest for students between 6 and 20 years old. During a 2 day expo all teams present their scientific or technical project to the public. WetenschapsExpo Sciences offers young students the opportunity to show their results to a wide audience, experts in the field and to their peers. In this way they do not only develop their scientific and technical knowledge, but also their presentation and communication skills.

The project

In this project we've created a Beowulf computer cluster from scratch. With old computers that were given by our school and some companies, we've created a cluster that was able to run parallel programed c++ programs (using MPICH).

What is a Beowulf Cluster?
A Beowulf Cluster is a specific type of computer cluster, where a set of computers is connected in a network (LAN) and set up to act as one pc. In this network, there’s one computer defined as the Main Node, the computer where the user actually gives his commands to the cluster. Next, there’s a Service Node, a computer which runs all the services necessary for the cluster, e.g. shared storage of files and handing out the right IP-address to each computer in the network. It’s also used as a firewall to protect the cluster against malware on the internet. A Worker Node is the last type of computer used in the cluster, which is only configured to do what is told by the Main Node.

Building the cluster
We started by disassembling the computers we got and mounting them on a wire rod. We also received some Dell computers, which we didn’t touch, because they were easy to stack eand the necessary hardware was already present. The only upgrade they’ve got is their memory (RAM). The ones with the best technical specifications were held apart to become Main or Service Node. The only problem we expected was the power consumption of our cluster: a normal pc uses a few hundred watts; we had multiple running, so we couldn’t use a normal power outlet. But the computers we use only consume about 100W, so we didn’t have a problem. Cooling the set also isn’t a problem, because they’re mounted in a 19” rack, which contains enough fans to cool every piece of hardware inside.

Installing Beowulf
After we finished the hardware, we started with the actual installation of the software onto the cluster. This has taken several weeks, because it’s very complex. All the computers have to communicate with each other and all execute one command at the same time. Afterwards, we made an image so anybody else could easily make his or her own cluster, just by installing it from our disc.
